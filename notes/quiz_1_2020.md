## A. Принцип Дирихле
```python
n = int(input())
m = int(input())
if m % n > 0:
    print(m // n + 1)
else:
    print(m // n)
```

## B. Сажени, аршины, пяди, вершки
```python
# 1 сажень = 3 аршин = 3 * 4 * 4 вершки = 48 вершки
# 1 аршин  = 4 пядь = 4 * 4 вершки = 16 вершки
# 1 пядь = 4 вершки

n = int(input())
s = n // 48
n -= s * 48
a = n // 16
n -= a * 16
p = n // 4
n -= p * 4
print(s, a, p, n)
```
